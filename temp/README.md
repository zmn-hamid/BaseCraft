# Base Craft - basic project for basic knowledged

Base Craft is a project made for those that need automation in their daily life
but don't know programming. This project is basically a startup project to
vibe-code on.

The reason I was motivated to create this project is I wanted to help 
non-programmers to develop their own basic tools and apps and reduce the time 
they waste on their mundane tasks. This does not mean one cannot build a 
complicated or huge app based upon this project, tho I don't advocate for this 
type of behavior myself.

## Prerequisities

- a laptop/pc
- 0 knowledge
- enough will to start developing :)

## The Basics

First you need to know some basics. Though you can potentially use agentic tools
like Hermes to do everything for you, a basic knowledge about how your tools
work would immensely help both speeding up your development and easing your 
mind about it, cause let's be honest; development is complicated.

Bellow is a list of basic tools and their tutorials. Feel free to skip the ones
you already know.

### Github - This Website!

The website you're reading this in, is called Github. It holds
different projects people have worked on. You can browse it and look up other
[open source](https://en.wikipedia.org/wiki/Open_source) projects (keep an eye on the licenses as well). The specifics of what github or git is, are not
important for the scope of this project. But there are two things you need to
know:

1. Each project is called a "repository" ("repo" in short).
2. To download the project you can simply click the green button at the top
called "**<> Code**", then in the drop-down click the last option called "**Download ZIP**". That's it.
    > "cloning" is kinda the same too. It's downloading the website via some
    other tools called **terminal** and **git**.

## How To Start

While I did mention there are no pre-requisities, you would benefit so much
from downloading a tool like **OpenCode** and setup your agent. Currently I am
using "**MimoCode**". It's free for now and you can just download and use. But for that you need to also have a very basic knowledge of **Command-Line** (also
called terminal, cmd, powershell or other tools). Agents like mimo can
automatically read your project and update it.
