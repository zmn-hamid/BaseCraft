# BaseCraft

**Tired of doing the same thing over and over? Let AI do it for you.**

You don't need to know programming. You don't need to learn code. Just tell AI what you want in plain English, and it writes the script for you. Then you double-click to run it. That's it.

---

## How It Works

```
You tell AI what you want
        ↓
AI writes the code
        ↓
You save the file
        ↓
You run it
        ↓
Task done!
```

**Example:** You say "I have 100 photos. Combine them into a 10x10 grid." AI writes the code, tells you where to save it, you double-click to run, and you get your grid.

---

## Quick Start

1. **Download** — Click the green `<> Code` button → "Download ZIP" → extract to Desktop
2. **Install** — Open the `BaseCraft` folder → double-click `install.bat` (or run `bash install.sh` on Mac/Linux)
3. **Upload** — Open an AI chatbot, upload `AGENTS.md` and `os.txt`, tell AI what you want

**New to terminals or file extensions?** Read the [full setup guide](docs/basic.md).

**Want more power?** Set up an [agent tool](docs/advanced.md) so AI works directly on your files.
