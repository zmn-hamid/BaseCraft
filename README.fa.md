# BaseCraft

**کارهای تکراری روزانه خود را به اسکریپت‌های ساده تبدیل کنید — بدون نیاز به دانش برنامه‌نویسی.**

BaseCraft یک پروژه شروع است که به شما کمک می‌کند ابزارهای خودکار کوچکی با استفاده از هوش مصنوعی بسازید. شما به زبان ساده انگلیسی توضیح می‌دهید چه می‌خواهید و هوش مصنوعی کد را برای شما می‌نویسد.

---

## شروع کار (۲ دقیقه)

### مرحله ۱: دانلود پروژه

1. به صفحه GitHub پروژه بروید
2. روی دکمه سبز "Code" کلیک کنید
3. روی "Download ZIP" کلیک کنید
4. فایل ZIP را در جایی آسان استخراج کنید (مثل دسکتاپ)

### مرحله ۲: نصب همه چیز

**ویندوز:**
1. پوشه `BaseCraft` را باز کنید
2. روی `install.ps1` دوبار کلیک کنید
3. اگر ویندوز پرسید "آیا می‌خواهید این اسکریپت را اجرا کنید؟"، روی "Open" یا "Run anyway" کلیک کنید
4. صبر کنید تا تمام شود

**مک/لینوکس:**
1. Terminal را باز کنید
2. تایپ کنید `cd ` سپس پوشه `BaseCraft` را به Terminal بکشید (این مسیر را برای شما تایپ می‌کند)
3. Enter بزنید
4. تایپ کنید `bash install.sh` و Enter بزنید
5. صبر کنید تا تمام شود

این کار Python و همه چیزهایی که نیاز دارید را نصب می‌کند. فقط یک بار این کار را انجام می‌دهید.

### مرحله ۳: سیستم عامل خود را به هوش مصنوعی بگویید

هوش مصنوعی برای کمک به شما نیاز دارد بداند سیستم عامل شما چیست.

1. پوشه `BaseCraft` را باز کنید
2. یک فایل متنی جدید به نام `os.txt` بسازید
3. یکی از اینها را داخل آن بنویسید: `windows`، `linux` یا `mac`
4. فایل را ذخیره کنید

تمام. آماده‌اید!

## چگونه کار می‌کند

1. **شما به هوش مصنوعی می‌گویید چه می‌خواهید** — به زبان ساده
2. **هوش مصنوعی کد را می‌نویسد** — آن را در پروژه شما ذخیره می‌کند
3. **شما آن را اجرا می‌کنید** — کار شما خودکار می‌شود!

### مثال: ترکیب تصاویر در یک شبکه

فرض کنید ۱۰۰ عکس دارید و می‌خواهید آنها را در یک شبکه ترکیب کنید.

**چه چیزی به هوش مصنوعی می‌گویید:**

> "من ۱۰۰ تصویر JPG در پوشه‌ای به نام 'photos' دارم. می‌خواهم آنها را در یک شبکه ۱۰x۱۰ ترکیب کنم. نتیجه را به عنوان 'grid.jpg' ذخیره کن."

**هوش مصنوعی چه کار می‌کند:**
- یک اسکریپت Python برای شما می‌نویسد
- یک اسکریپت ساده برای اجرای آن می‌سازد

**شما چه کار می‌کنید:**
- عکس‌هایتان را در پوشه `workspace` قرار دهید
- اسکریپتی که هوش مصنوعی ساخته را اجرا کنید
- تصویر شبکه خود را بگیرید!

## اجرای اسکریپت‌ها

وقتی هوش مصنوعی اسکریپتی برای شما می‌نویسد، یک اسکریپت "اجرا" ساده در پوشه اصلی ایجاد می‌کند.

فقط روی آن اسکریپت دوبار کلیک کنید تا اجرا شود. تمام.

اگر می‌خواهید اسکریپتی را دستی اجرا کنید، Terminal را در پوشه BaseCraft باز کنید و تایپ کنید:

```
uv run python your_script.py
```

## فضای کاری شما

پوشه `workspace` جایی است که فایل‌هایتان را برای پردازش توسط هوش مصنوعی قرار می‌دهید.

```
BaseCraft/
├── workspace/          # فایل‌هایتان را اینجا قرار دهید
├── scripts/            # اسکریپت‌های کمکی (دست نزنید)
├── install.ps1         # نصب‌کننده ویندوز
├── install.sh          # نصب‌کننده مک/لینوکس
├── os.txt              # سیستم عامل شما (این را بسازید)
└── README.md           # این فایل
```

## نکاتی برای صحبت با هوش مصنوعی

هر چه بهتر توضیح دهید چه می‌خواهید، نتیجه بهتر می‌شود.

- **مشخص باشید**: "همه تصاویر را به ۸۰۰x۶۰۰ پیکسل تغییر اندازه بده" بهتر از "تصاویر را کوچک‌تر کن" است
- **نوع فایل‌ها را بگویید**: به هوش مصنوعی بگویید فایل‌های شما چه فرمتی هستند (JPG, PNG, CSV و غیره)
- **خروجی را توصیف کنید**: نتیجه باید چه شکلی باشد؟ کجا ذخیره شود؟
